##Installation
>webpack 4+ 需要再安裝 webpack-cli

新建webpack專案
```sh
mkdir webpack-demo
cd webpack-demo
npm init -y
npm i -D webpack-cli webpack
```

安装 beta 版本
```sh
npm install --save-dev webpack@next
# 或特定的 tag/分支
npm install --save-dev webpack/webpack#<tagname/branchname>
```


### Bundle

```sh
npx webpack
```
